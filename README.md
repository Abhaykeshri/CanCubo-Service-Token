## 🌟 Overview

**CanCubo Service Token (CCST)** is a Solana-based SPL utility token designed with a strong security model using **multi-signature mint authority** and **on-chain metadata** via Metaplex.

It is deployed on **Solana Devnet** and intended for testing governance, treasury control, and service-based token utilities.
